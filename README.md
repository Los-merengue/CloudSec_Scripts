## Configuration

Create a `.env` file in the root directory with the following contents:

- DATABASE_URL=
- SMTP_USERNAME=
- SMTP_PASSWORD=
- SMTP_SERVER=
- SMTP_PORT=
- SENDER_EMAIL=

Replace the values with your actual database credentials and SMTP settings.
